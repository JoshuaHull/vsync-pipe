# vsync pipe

Efficient reactivity in Angular without Observables. Combines Vue's reactivity API with the async pipe.

## Installation

> npm install vsync-pipe

With peer dependencies:
  * @angular/common
  * @vue/reactivity
  * @vue/runtime-core
  * rxjs
